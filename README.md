# 站长联系方式网站

这是一个功能完善、设计精美的站长联系方式网站，旨在为网站访问者提供多种便捷的联系方式，帮助站长与访客建立良好的沟通渠道。

## 项目特性

### 完整的页面结构
- 响应式导航栏，支持移动端菜单
- 引人注目的英雄区域，突出网站主题
- 关于站长介绍，展示个人信息
- 多样化联系方式展示
- 功能完善的联系表单
- 社交媒体链接集成
- 常见问题解答(FAQ)折叠面板
- 信息丰富的页脚区域

### 核心功能

#### 1. 联系方式展示
- 电子邮箱：支持直接点击发送邮件
- 电话号码：支持直接点击拨打电话
- 办公地址：清晰展示地理位置信息
- 工作时间：提供业务咨询时段
- 微信：支持扫码添加好友
- QQ：提供快速联系链接

#### 2. 联系表单
- 姓名、电子邮箱、主题、消息等完整字段
- 实时表单验证，确保数据有效性
- 提交状态反馈，成功/失败提示
- 表单重置功能

#### 3. 响应式设计
- 桌面端、平板和移动设备完美适配
- 针对不同屏幕尺寸优化的布局
- 触控友好的交互元素
- 自适应导航栏

#### 4. 动画与交互
- 滚动触发的渐入动画
- 元素悬停效果和过渡动画
- 按钮点击反馈
- 平滑滚动体验
- 表单交互反馈

## 技术栈

- **HTML5**：语义化页面结构
- **CSS3/Tailwind CSS**：现代化样式和响应式设计
- **JavaScript**：交互功能和表单处理
- **Font Awesome**：图标库支持
- **响应式设计**：多设备兼容

## 如何使用

### 安装与部署

1. 克隆或下载项目文件到本地
2. 修改配置中的实际联系方式和个人信息
3. 将项目文件部署到您的Web服务器

### 本地开发测试

使用Python简单HTTP服务器进行本地测试：

```bash
cd /path/to/project
python -m http.server 8000
```

然后在浏览器中访问 http://localhost:8000

### 自定义配置

1. 修改`index.html`中的个人信息和联系方式
2. 替换`images/`目录中的示例图片
3. 根据需要调整`styles.css`中的样式
4. 修改`script.js`中的表单提交处理逻辑

## 响应式设计

网站在以下设备上均有优化的显示效果：
- 桌面端：大于1200px
- 平板横向：992px - 1199px
- 平板纵向：768px - 991px
- 移动设备：小于767px

## 表单处理

当前表单使用客户端验证和模拟提交。要实现真实的表单提交功能，您需要：

1. 设置后端处理脚本
2. 修改`script.js`中的表单提交逻辑
3. 配置适当的服务器端验证
